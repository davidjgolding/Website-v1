ME.

I’m a first-year Computer Science student at the University of Warwick. Outside of term time, I’m located in Sleaford, Lincolnshire.

I enjoy software development and the challenges that it presents. I have experience using and developing programs in a variety of languages, including Java and Python and I'm proficient in the object-oriented paradigm. This includes in the creation of client-server applications, APIs and graphical user interfaces (GUIs). I also have an interest in web development.

Warwick has supercharged my understanding of Computer Science. It's reinforced the problem-solving mindset required in order to solve complex problems and introduced me to new algorithms. This has allowed me to solve problems which delivers the best compromise on speed and efficiency using the likes of AVL trees and hashing. Finally, by being taught from first-principles has meant that I'm able to easily pick up new languages as all there really is to learn is the syntax.

At the heart of any front-end application, I create is the user. You can have a great backend behind the interface but if the experience for the user is poor, then the application is destined for failure. I always aim to create easy to use and aesthetically pleasing applications.

In terms of applications for my skills, I have wide-ranging interests: from financial markets to autonomous vehicles. Any way in which I can apply technology to improve an experience I'm interested in.
